# PRODUCT — FounderNexus Mockup Feedback Surface

## Product Summary

Agent-agnostic visual feedback tool for mockups.

## Users

- Team members and reviewers providing content/UI feedback
- Site owner consolidating and actioning feedback

## Problem It Solves

Quick and easy UI annotations on mockups, with sharing back to the owner.

## Tech Stack

- TypeScript/JavaScript
- HTML/CSS
- Static GitHub Pages deployment

## Current Status

Adding native mockup commenting and share workflow (pin comments, export/import, shareable payload link).
